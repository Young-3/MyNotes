

# IDEA导入项目中文乱码

先点右下角reload为GBK

然后convert为UTF-8

# 设置idea隐藏项目中的 没用的配置文件 

Ctrl+Alt+s--> Editor --> FileTypes   或者 File --> Setting --> Editor --> FileTypes

找到Ignore files and folders 编写隐藏文件 配置;

要注意的是:  每一个要隐藏的 后缀 后面要加一个  分号( ; )

例如:    *.idea;*.iml;
